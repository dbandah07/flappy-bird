# Flappy Bird (Unity Lab, C#)
Mechanics-focused Unity lab project recreating core *Flappy Bird* gameplay. Built to practice physics tuning, obstacle spawning, and score tracking.

## Features
- Physics-based flap movement
- Obstacle spawning with consistent spacing
- Progressive difficulty scaling:
  - Increased forward speed over time
  - Pipes move further apart as speed increases
  - Flap strength scales to match increased speed
- Score triggers and restart loop
- Controls:
  - Desktop: Keyboard input (as implemented in the lab)
  - Mobile: Tap input (as implemented in the lab)

## Notes
Completed during Fall 2025 as part of the class **Mobile Game Development**
